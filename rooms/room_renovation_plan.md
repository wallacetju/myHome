# 房间布局改造建议

> 目标：迎接6月新生儿，将bedroom2改造为婴儿房，整体提升空间利用率

---

## 一、整体现状分析

**核心问题：功能区混乱 + 物品堆积**

| 房间 | 现状 |
|------|------|
| 客厅 | 办公区侵占客厅空间，桌面杂物多，家具体积较大 |
| 书房 | **重灾区**：地面堆满衣物，几乎无法正常使用 |
| 卧室1 | 开放衣架+多个行李箱占据大量空间 |
| 卧室2 | 有床和储物柜，杂物较多，需清空改造 |
| 衣帽间 | 相对整洁，仍有收纳潜力可挖掘 |
| 卫生间 | 台面略杂乱，总体尚可 |

---

## 二、Bedroom2 婴儿房改造方案

### 需要清空的物品
- [ ] 现有的床（可移至书房或处理）
- [ ] 墙角杂物和储物袋
- [ ] 散落的衣物和包装

### 婴儿房基本配置
- [ ] 婴儿床（靠墙放置，远离窗户和空调直吹位置）
- [ ] 尿布台/换衣台（带储物功能）
- [ ] 小型衣柜或斗柜（存放婴儿衣物用品）
- [ ] 哺乳椅（方便夜间喂奶）
- [ ] 柔和灯光（小夜灯，方便夜间照顾）

### 安全注意事项
- 窗帘绳收好，避免缠绕风险
- 插座安装保护盖
- 家具固定防倾倒
- 避免放置小件物品和毛绒玩具（窒息风险）

---

## 三、建议处理/丢弃的物品清单

| 物品 | 位置 | 建议 | 优先级 |
|------|------|------|--------|
| 欧式沙发 | 客厅 | 考虑更换为简约款，体积庞大且风格厚重 | 中 |
| 厚重茶几 | 客厅 | 换成轻便小茶几或边几 | 中 |
| 开放式衣架 | 卧室1 | 衣物转移至衣帽间后，衣架处理掉 | 高 |
| 行李箱（多个）| 卧室1顶部 | 只保留1-2个常用的，其余处理 | 高 |
| 书房地面衣物 | 书房 | 断舍离！筛选后捐赠或丢弃 | **最高** |
| 健身车 | 客厅窗边 | 如不常用，建议处理 | 中 |
| 小冰箱 | 电视柜旁 | 如非必需，可移除 | 低 |
| 过多的靠垫/毯子 | 客厅沙发 | 精简至2-3个 | 低 |
| 不再阅读的书籍 | 书房书柜 | 捐赠或二手出售 | 中 |

---

## 四、功能区重新规划

### 方案：将办公区从客厅移至书房

#### 第一步：书房大清理
- [ ] 清空地面所有衣物杂物
- [ ] 书柜内书籍精简（不再看的书可捐赠）
- [ ] 清理角落堆积物
- [ ] 放置一张简洁办公桌 + 电脑设备

#### 第二步：客厅还原为纯休息区
- [ ] 移除办公桌和显示器设备
- [ ] 只保留沙发、茶几、电视区
- [ ] 清理桌面杂物
- [ ] 为宝宝预留地面活动空间（未来爬行垫区域）

---

## 五、收纳优化建议

### 衣帽间充分利用
- 卧室1的开放衣架衣物全部转入衣帽间
- 按季节/类别整理，使用统一衣架

### 统一收纳系统
- 购买同款收纳盒，杂物分类装箱
- 贴上标签，方便查找
- 视觉上更整洁有序

### 垂直空间利用
- 墙面搁板存放展示品
- 门后挂钩挂包包/帽子
- 床底收纳箱存放换季物品

### 日常习惯
- 执行"一进一出"原则：添置新物品时，同步清理旧物
- 每周固定时间简单整理，避免堆积

---

## 六、执行优先级排序

| 顺序 | 任务 | 备注 |
|------|------|------|
| 1 | 立即清理书房 | 恢复可用状态 |
| 2 | 办公设备从客厅搬至书房 | 释放客厅空间 |
| 3 | 清空bedroom2 | 为婴儿房腾空 |
| 4 | 卧室1衣物整理 | 转移至衣帽间 |
| 5 | 婴儿房布置 | 购买婴儿家具用品 |
| 6 | 客厅家具精简 | 可在宝宝出生前逐步完成 |

---

## 七、时间规划建议

| 时间 | 目标 |
|------|------|
| 2月 | 书房清理 + 办公区搬迁 |
| 3月 | 卧室1整理 + bedroom2清空 |
| 4月 | 婴儿房基础布置 |
| 5月 | 细节完善 + 安全检查 |
| 6月 | 迎接宝宝 |

---

## 备注

- 宝宝出生后需要在家中频繁走动（喂奶、换尿布、哄睡），保持动线畅通非常重要
- 断舍离是关键：不用的东西只会占据空间和精力
- 可考虑使用闲鱼等平台出售二手物品，既环保又能回收部分资金
