## File Formats
*   **HEIC:** High Efficiency Image Container. These are likely original photos taken with a modern smartphone (e.g., iPhone).
*   **JPG:** Standard JPEG images, likely used as previews or compatible versions of the HEIC files.

## Usage
*   **Viewing:** Use compatible image viewers to open HEIC files (e.g., Preview on macOS).
*   **Conversion:** The HEIC files may need to be converted to JPG or PNG for broader compatibility with web browsers or other software.
